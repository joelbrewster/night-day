# night-day app

## Requirements
- You will need 2 wallpaper pictures in: `~/Pictures`
  - day.png
  - night.png
- You will need 2 themes in Terminal.app named:
  - day
  - night

## How to use
Start night-day app to change theme, then quit it (cmd+q)
